# amxpl
Arduinoを介した物理機器でXplaneの操作を行うためのAir Managerコード  

### requirements
x-plane(zibo 738X or Aircraft equipped with g1000)  
air manager  
arduino mega  
dual axis rotery encoder  
single axis rotery encoder  
slider switch for throttle
